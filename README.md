# KsWebExtension

## Quick Guide

1. **Update Version**: Change the version in both `manifest.json` and `package.json`.

2. **Build**: Ensure `kstable.js` is updated if needed.

3. **Package**: Zip `manifest.json`, `inject.js`, and `kstable.js`.

4. **Upload**: Go to the Firefox Add-ons Developer Hub, submit the new version zip, and follow the prompts.

5. **Tag Release**: After approval, tag the version in Git.

That’s it—happy deploying!
